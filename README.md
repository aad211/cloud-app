# CLOUD App

**CLOUD** - Cough Lung Observation & Diagnosis

A Flutter application for cough analysis and lung health monitoring.

## Getting Started

This is a Flutter project for iOS, Android, and Web platforms.

### Prerequisites

- Flutter SDK ^3.10.3
- Dart SDK (bundled with Flutter)

### Running the App

```bash
# Get dependencies
flutter pub get

# Run on your device/emulator
flutter run

# Run on web
flutter run -d chrome
```

### Building

```bash
# Android
flutter build apk

# iOS (requires macOS)
flutter build ios

# Web
flutter build web
```

### Testing

```bash
# Run all tests
flutter test

# Run with coverage
flutter test --coverage

# Analyze code
flutter analyze
```
